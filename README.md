# vite 创建 vue3+ts 项目

[vite创建vue项目]: https://vitejs.cn/vite3-cn/guide/#scaffolding-your-first-vite-project

```bash
pnpm create vite 3vite-elementplus-vue3-admin --template vue-ts

# 安装依赖
pnpm install

# 启动项目
npm run dev
```
